# Sniper Tools

Professional calculators for precision shooting.

🔗 **Live Demo:** [jjllrrvvrr.github.io/SniperTools](https://jjllrrvvrr.github.io/SniperTools/)

## Tools

- **Loophole** - Calculate bullet trajectory through obstacles
- **Range Finder** - Estimate distance using mil-dot measurements
- **MGRS Calculator** - Distance and bearing between grid coordinates
- **Unit Converter** - Metric/imperial conversions
- **Spotter Test** - Correction calling training
- **Judging Distance** - Distance estimation practice
- **Weather** - Current conditions and data
- **Useful Links** - Reference resources

## Features

- Dark/Light mode
- Mobile responsive
- Works offline
- No tracking
